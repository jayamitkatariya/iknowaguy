**AI agents bring humans into the loop**

> **Note:** Not yet published to npm. Run from source.

## Run from source

```bash
cd packages/cli
pnpm build
node bin/run init
```

## Commands

```bash
iknowaguy init             # Initialize ‚ÄĒ create project config
iknowaguy dev              # Start dev ‚ÄĒ run everything locally
iknowaguy setup:agent      # Link agent ‚ÄĒ connect Hermes/OpenClaw/Claude
iknowaguy setup:notify     # Notifications ‚ÄĒ Slack, Telegram, Email, SMS
iknowaguy setup:payments   # Payments ‚ÄĒ Stripe, PayPal, Manual
iknowaguy config           # Show config ‚ÄĒ view current settings
iknowaguy doctor           # Diagnose ‚ÄĒ check setup health
```

## Quick Start

```bash
cd packages/cli
pnpm build
node bin/run init
node bin/run dev
node bin/run doctor
```

## Development

```bash
pnpm build   # Build CLI
pnpm dev     # Build and run
```

## Features

- **MCP-first** ‚ÄĒ 10 MCP tools for AI agent integration
- **Multi-agent** ‚ÄĒ Hermes Agent, OpenClaw, Claude Code supported
- **Notifications** ‚ÄĒ Slack, Telegram, Email, SMS via unified adapter
- **Payments** ‚ÄĒ Stripe, PayPal, or Manual (bounty payouts)
- **Docker** ‚ÄĒ Full stack with `docker compose up`
