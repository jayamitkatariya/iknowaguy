**AI agents bring humans into the loop**

## Install

```bash
npm install -g hireahuman
```

## Commands

```bash
hireahuman init             #  INITIALIZE ‚ÄĒ create project config
hireahuman dev              #  START DEV ‚ÄĒ run everything locally
hireahuman setup:agent      #  LINK AGENT ‚ÄĒ connect Hermes/OpenClaw/Claude
hireahuman setup:notify     #  NOTIFICATIONS ‚ÄĒ Slack, Telegram, Email, SMS
hireahuman setup:payments   #  PAYMENTS ‚ÄĒ Stripe, PayPal, Manual
hireahuman config           #  SHOW CONFIG ‚ÄĒ view current settings
hireahuman doctor           #  DIAGNOSE ‚ÄĒ check setup health
```

## Quick Start

```bash
# 1. Initialize your project
cd my-ai-agent-project
hireahuman init

# 2. Paste the SQL in Supabase SQL Editor
# ‚Üí supabase/migrations/001_complete.sql

# 3. Start everything locally
hireahuman dev

# 4. Link your AI agent
hireahuman setup:agent

# 5. Check health
hireahuman doctor
```

## Development

```bash
npm run build   # Build CLI
npm run dev     # Build and run
```

## Features

- **MCP-first** ‚ÄĒ 10 MCP tools for AI agent integration
- **Multi-agent** ‚ÄĒ Hermes Agent, OpenClaw, Claude Code supported
- **Notifications** ‚ÄĒ Slack, Telegram, Email, SMS via unified adapter
- **Payments** ‚ÄĒ Stripe, PayPal, or Manual (bounty payouts)
- **Docker** ‚ÄĒ Full stack with `docker compose up`
