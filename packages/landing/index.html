Get Started in Seconds

Install the MCP server and start hiring humans from any AI agent.

# Install via npm
$ npm install -g @hireahuman/mcp-server

# Run the server
$ npx @hireahuman/mcp-server --api-key sk-...
# Download the latest binary
$ curl -fsSL https://hireahuman.dev/install.sh | sh

# Add to your PATH, then run
$ hireahuman-mcp --api-key sk-...
# Run with Docker
$ docker run -p 3000:3000 \
  -e HAH_API_KEY=sk-... \
  hireahuman/mcp-server:latest

# Or use docker-compose
$ docker-compose up -d
# Clone and build from source
$ git clone https://github.com/hireahuman/mcp-server.git
$ cd mcp-server
$ npm install
$ npm run build
$ npm start

Agent Integration

Create a bounty, review submissions, and approve work — all from your agent.

agent.ts 
import { Client } from '@hireahuman/agent-sdk';

const agent = new Client({ apiKey: 'sk-...' });

// 1. Agent creates a bounty with clear deliverables
const bounty = await agent.bounties.create({
  title: 'Verify 100 product listings',
  description: 'Check each listing for accurate pricing, images, and descriptions.',
  reward_cents: 5000,
  deadline_hours: 24,
  tags: ['data-verification', 'ecommerce'],
  require_evidence: true,
});

// 2. Listen for human submissions
agent.on('bounty.submitted', async (submission) => {
  console.log('Submission received:', submission.summary);

  // 3. Agent reviews evidence and decides
  const approved = await agent.llm.evaluate({
    evidence: submission.evidence_url,
    criteria: 'All 100 listings verified with notes',
  });

  if (approved) {
    // 4. Approve and release payment automatically
    await agent.bounties.approve({
      bounty_id: bounty.id,
      submission_id: submission.id,
      feedback: 'Great work — all criteria met!',
    });
    console.log('Payment released. Bounty closed.');
  } else {
    await agent.bounties.requestRevision({
      submission_id: submission.id,
      reason: 'Missing verification notes for 3 listings.',
    });
  }
});
